# dsk_filter

A simple thing for filtering `simplified DSK tables`.

## Quick Setup

```sh
# clone the project
git clone https://github.com/moosetheory/dsk_filter.git

# enter the project directory
cd dsk_filter

# install dependency
npm install

# develop
npm run dev
```
